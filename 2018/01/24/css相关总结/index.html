平时的一些总结

CSS 中类 (classes) 和 ID 的区别

一个元素只能有一个id属性，但可以有多个class属性；
id的样式优先级高于class；
id在文档中只能使用一次，class可以使用很多次；

“resetting”和“normalizing”css的区别

为什么会有默认样式重置的出现？当我们没有给html元素设置样式时，浏览器会根据自己默认样式对html元素布局，但每个浏览器默认样式不同，就会导致同一个页面在不同浏览器上面显示不同。
Resetting是重新定义标签样式，来覆盖浏览器的默认属性
Normalizing 保护有用的浏览器默认设置而不是完全去掉他们

二者区别：normalizing.css保护了有价值的默认值、修复了常见的移动端和桌面端浏览器的bug、不会让调试工具变得混乱。并且normalizing.css有详细的文档、是模块化的，可以很清楚的看到哪些部分被设置了特定的值，因此可以有选择性的移除掉一些永远不会用的部分。
reset和normalize如何选用？如果用到reset.css 要选择性的使用。
如使用normalizing.css可以把它当作自己项目的基础css，自定义样式值来
满足自己要求自己要求。

请解释浮动 (Floats) 及其工作原理。

浮动元素会脱离标准文档流，浮动于标准文档流之上。
浮动的危害：会使父元素坍塌，因为父元素在高度计算的时候会忽略到浮动元素。
清除浮动的几种方法：
1、结尾处加空div标签clear：both。
2、父级div定义伪类:after和zoom。
3、父级div定义overflow：hidden。
4、BFC。
5、父级div也加个浮动

盒模型

包括外边距、边框、内边距和元素内容。一般我们定义的都是元素内容的宽高。通常设置的背景显示区域是内容、内边距和边框这一块。外边距是透明的不会遮挡其他元素。
元素框的总宽度 = 元素（element）的width + padding的左边距和右边距的值 + margin的左边距和右边距的值 + border的左右宽度；
元素框的总高度 = 元素（element）的height + padding的上下边距的值 + margin的上下边距的值 ＋ border的上下宽度。
CSS外边距合并。
box-sizing 属性：content-box/border-box/inherit

伪元素和伪类

伪元素：不是真正的页面元素，是由css样式展示的行为。如::before、::after、::first-line、:first-letter；其中::before和::after添加的内容默认是inline元素。伪元素不属于文档，所以js不能操作。
伪类：关于<a>，常见的有五个伪类:link、:visited、:focus、:hover、:active。UI元素伪类包括：:enabled、:disabled、:checked，主要针对form元素，IE8不支持。

BFC

https://www.cnblogs.com/lhb25/p/inside-block-formatting-ontext.html
box formatting context(块级格式化上下文)：是一个独立的渲染区域，只有block-level box，规定了内部的block-level box如何布局，并且与外部无关
会生成BFC的元素:1、根元素。2、float属性不为none。3、position属性为absolute或fixed。4、display为inline-block,table-cell,table-caption,flex,inline-flex。5、overflow不为visible
BFC布局规则：
1、计算BFC高度时，浮动元素也会计算在内。（例如给父元素设overflow:hidden可以防止子元素是浮动布局撑不起父元素）
2、BFC区域不会与float box重叠（多栏布局的一种方式）
3、BFC就是页面上一个隔离的独立容器，容器里面的子元素不会影响到外面元素，反之也是。
4、box垂直方向的距离由margin决定，同一个BFC内的box的margin会发生重叠。（可以用新建BFC来防止margin重叠）

CSS sprites

css sprites其实就是把网页中的一些图标整合到一张图片文件中，在利用css的background-image 、background-repeat 、background-position进行背景定位。

图片替换方法

http://blog.csdn.net/cwzhsi/article/details/46403077

图片替换方法主要是指将文字替换成图片的技术，即在html中使用文字，浏览器显示时用对应的图片显示。意义在于做网站优化，文字才是搜索引擎主要搜索对象。
1、FIR替换法。

把span标签里的文字display：none。指定h1 的背景图片。这个方法的问题在于一些流行的屏幕阅读器会忽略那些display为none或者visibility为hidden的元素，因此会完全忽略掉这个文本。且若关闭浏览器显示图片同时启用css支持时，文字图片都不显示。
2、phark。设置文字text-indent为很大的负值，把文字显示在屏幕以外。这个不会影响屏幕阅读器的使用。但若关闭浏览器显示图片同时启用css支持时，文字图片都不显示。
3、NIR。利用伪元素和伪元素产生的内容来实现图像替换（但ie有的版本不支持伪元素）。
